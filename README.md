## 📌 Project Overview

The **Piyush E-commerce Sales Dashboard** is an interactive data visualization project developed to analyze online sales performance and generate meaningful business insights. The dashboard helps in understanding customer behavior, profit trends, sales performance, and product analytics.

This project demonstrates the use of **data analytics and visualization techniques** to support data-driven business decisions.

---

## 🚀 Key Features

✅ Sales & Profit Analysis  
✅ State-wise Sales Performance Insights  
✅ Customer & Category Analysis  
✅ Monthly Profit Trend Analysis  
✅ Payment Mode Distribution  
✅ Product & Sub-Category Insights  
✅ Interactive Visualizations & Filters  
✅ Clean and Professional Dashboard Design
